CMintS starter

Launch and host your wesbite in no time

  • Install NodeJS if you don't have it yet.
  • Fork Current repository.
  • Clone or download the forked repository and navigate to that directory
  • run current command
npm i
npm run dev
  • Navigate to http://127.0.0.1:3000
  • Hurray you are running current website locally! Now let's make some changes.
  • Update the website the way you want. You can learn more about CMintS here
  • Now when you are done editing let's publish the website
  • From the project root directory run following command
npm run deploy
